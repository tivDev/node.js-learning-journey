# OS Module

This folder demonstrates how to use the Node.js `os` module to access system-level information.

## Topics Covered

- System information (type, platform, arch, release)
- CPU details
- Memory usage
- User info
- System uptime

## Run Examples

```bash
node system-info.js
node cpu-info.js
node memory-info.js
node user-info.js
node uptime.js
